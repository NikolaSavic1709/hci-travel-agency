Stay Eat page

Stay Eat page is used for displaying Stays and Restaurants, you can search Stays and Restaurant by name, display their location by clicking the map, or add a new Stays and Restaurant.

These cards look a bit different, the headers displays the type. When the eye butten is clicked the details are opened

This window allows you to add a view restaurant or a hotel (amenities are only present when viewing a hotel), for addtional help refer to help page for this window.

Stay Eat page

Stay Eat page is used for displaying Stays and Restaurants, you can search Stays and Restaurant by name, display their location by clicking the map, or add a new Stays and Restaurant.

These cards look a bit different, the headers displays the type. When the eye butten is clicked the details are opened

This window allows you to add a view restaurant or a hotel (amenities are only present when viewing a hotel). Title, Location and Description are all text fields, Coordinates are added by searchin whithin the search field on the map, or by clicking on a location on map, a pin will be place that coresponds with selected location.
Amenities are only present when Stay is selected in combo box, adding amenities is done by using drag and drop, or add all button. Now confirm the selection and you successfully created a new Stay/Restaurant.

Navbar

Navigation is done through the navbar on the top of the page. The active page is highlighted white, while other pages are green with white text. Navbar offers these options:

  • Home - homepage is the startup page and it host a route overview
  • Places - this page hosts all available attractions
  • Stay&Eat - this page hosts all available hotels and restaurants
  • Report - this page hosts a monthly report for all available routes
  • History - this page host all history of all transactions
  • Help - it opens this window (self explanatory unless you got here by pressing F1
  • Logout - logs you out of the app

General elements of interaction

Here we will go over some general elements of interaction

The main medium for data display are the cards and all of the Attractions routes and everything will be displayed with cards. On click of the card additional data vill be displayed on map.

All cards contain these 2 elements that are shown when the cards are hovered. Eye - on click shows more details Bin - on click deletes item

Other important element of the interaction is map, this displays location or routes and allows of picking a location when creating an item.

If available this button opens dialog for creating a new item (route/attraction/.. depending on the current page)

This allows to search items by name or filter them by some cattegory (according on the data that is being filtered)

Filter example: here we can search or filter by price or number of people
